# Student Performance Dashboard (Power BI)

## Project Overview
This project analyzes student performance data using Power BI.

## Features
- Average Maths Marks KPI
- Average English Marks KPI
- Student Performance Distribution (Pie Chart)
- Student Marks Comparison (Bar Chart)
- Interactive Result Filter (Slicer)

## Tools Used
- Power BI Desktop
- CSV Dataset
